#1 Поліграф як різновид спеціального психофізіологічного технічного засобу

Визначення: Поліграф — багатоканальний електрофізіологічний прилад для одночасної реєстрації фізіологічних параметрів організму у відповідь на психологічні стимули (запитання).

Офіційний статус: НЕ «детектор брехні» — він реєструє не брехню, а психофізіологічні реакції, пов'язані зі стресом під час обману.

Канали реєстрації:

  • Кардіоваскулярний — артеріальний тиск та пульс (ЧСС, форма пульсової хвилі).
  • Дихальний — грудне та черевне дихання (глибина, частота, ритм). Два пневмографи.
  • ЕДА (електродермальна активність) — шкірно-гальванічна реакція, потовиділення.
  • ФПГ (фотоплетизмограф) — периферійний кровообіг, судинний тонус пальця.

Апаратна схема:

  • Датчики → підсилювачі → АЦП → комп'ютер з програмним забезпеченням.

Сучасні системи: Lafayette LX5000, Stoelting Ultrascribe, Axciton, Limestone Technologies — аналогічна архітектура, різне ПЗ аналізу.

⚠️ УВАГА: Поліграф НЕ вимірює брехню — він вимірює симпатичне збудження нервової системи. Правильний висновок робить ФАХІВЕЦЬ, а не прилад.

#7 Теоретичні концепції діагностики брехні. Брехня як психологічна категорія

Брехня: Навмисне спотворення інформації з метою введення іншої особи в оману. Ключова умова — УСВІДОМЛЕНІСТЬ.

Теорії:

  • Страху викриття (Backster, Reid): страх бути викритим → активація симпатичної НС → вимірювані зміни.
  • Почуття провини: конфлікт між дією і моральними нормами → тривога → реакція.
  • Когнітивного навантаження: брехня потребує більших когнітивних зусиль → розходження реакцій.
  • Прихованого знання: злочинець знає «секрет» → орієнтовний рефлекс на пов'язані стимули.

Фізіологічна картина при брехні:

  • ЧСС: прискорення, збільшення ударного об'єму серця.
  • Дихання: порушення ритму, апное або глибокий видих.
  • ЕДА: різке збільшення шкірної провідності (потовиділення).
  • ФПГ: зменшення амплітуди (вазоконстрикція — судини звужуються).
⛔ КРИТИЧНО: Реакції НЕСПЕЦИФІЧНІ — однакові при різних емоціях, хворобах, страху. Тому потрібні контрольні запитання для диференціації.

#30 Поняття поліграфологічного методу

Метод: Теоретично обґрунтована система принципів виявлення обману на основі психофізіологічних реакцій. Визначає логічну структуру дослідження.

Класифікація методів:

  • CQT (Метод контрольних запитань): порівняння реакцій на релевантні та контрольні. Найпоширеніший.
  • CIT (Метод прихованої інформації): виявлення реакції на деталі, відомі лише злочинцю. Найбільш науково обґрунтований.
  • POT/SPOT (Метод піку напруженості): пошук пікової реакції серед альтернатив.
  • DLST (Метод керованого обману): порівняння «еталону обману» з реакціями на релевантні запитання.

Метод vs Методика: Метод — концепція. Методика — конкретна технічна реалізація з алгоритмом. Один метод → кілька методик.

🎯 4 методи поліграфології
**CQT** (контрольні) → **CIT** (приховане) → **POT** (пік) → **DLST** (керований обман)

#39 Поняття поліграфологічної методики

Методика: Конкретна технічна реалізація методу з деталізованим алгоритмом: структура запитань, тип контрольних, кількість циклів, система оцінювання, правила прийняття рішення.

Основні методики:

  • AFMGQT — Модифікована методика загальних запитань ВПС США. Найпоширеніша.
  • Utah ZCT — Тест порівняння зон Університету Юти. Науково найвалідніша.
  • Federal ZCT — Федеральний тест порівняння зон. «Золотий стандарт» США.
  • Federal You-Phase — Федеральна методика «Фаза-Ви» (акцент «особисто»).
  • DLST — Скринінговий тест керованого обману.
  • TES — Тест на шпигунство і саботаж.
  • Методика Раскіна (Utah Four-Question Test) — Historична, основа сучасних.
ℹ️ ПРИМІТКА: Вибір методики залежить від мети (скринінг vs діагностика), часу та кваліфікації фахівця.

#73 Види поліграфологічних досліджень

  • Скринінгові: великі групи, широкі категорії, швидкий формат (DLST). Кадрові перевірки.
  • Діагностичні: конкретний інцидент, деталізовані запитання (AFMGQT, ZCT). Кримінальні справи.
  • Дослідження прихованої інформації (CIT): чи обізнана особа з деталями події.

За регулярністю:

  • Разові — в рамках конкретного розслідування.
  • Планові/Периодичні — 1–2 рази на рік для діючих співробітників.

За формою:

  • Індивідуальні (стандарт) та Групові (масові кадрові перевірки).

#2 Модифікована методика загальних запитань ВПС США (AFMGQT)

Повна назва: Air Force Modified General Question Technique.

Принцип: Порівняння реакцій на РЕЛЕВАНТНІ (про інцидент) та КОНТРОЛЬНІ (про минулі схожі дії) запитання.

Механізм диференціації:

  • Невинуватий: більше реагує на КОНТРОЛЬНІ (найбільш загрозливі для нього).
  • Винуватий: більше реагує на РЕЛЕВАНТНІ (реальна загроза викриття).

Структура циклу:

  • Н1, Н2 — нейтральні (розминка). К1 — контрольне. Р1 — релевантне. К2 — контрольне. Р2 — релевантне. С — симптоматичне (опційно).
  • Циклів: 3–4. Загальний час з бесідами: 45–90 хвилин.
ℹ️ ПРИМІТКА: AFMGQT — НАЙПОШИРЕНІША методика в країнах НАТО. Вимагає якісної передтестової бесіди та ретельних контрольних запитань.

#4 Скринінговий тест запитань керованого обману (DLST)

Повна назва: Directed-Lie Screening Test.

Принцип: Досліджуваному НАКАЗУЮТЬ навмисно неправдиво відповісти на конкретне запитання. Ця задокументована відповідь стає «еталоном обману» для порівняння.

Алгоритм:

  • 1. Пояснення суті керованого обману.
  • 2. Моделювання реакції на навмисну брехню.
  • 3. Запис індивідуального «підпису обману».
  • 4. Порівняння з реакціями на релевантні запитання.

Переваги:

  • Не потребує тривалої розробки контрольних запитань.
  • Швидкий формат: 5–15 хв на тест. Ефективний для масового скринінгу.
  • Більш об'єктивний (менша суб'єктивність фахівця).

Недоліки:

  • Деякі досліджувані психологічно не можуть навмисно обманювати.
  • Менш ефективний для детальних кримінальних розслідувань.
ℹ️ ПРИМІТКА: DLST — основна методика для масового скринінгу та кадрових перевірок.

#16 Тест піку напруженості відомого рішення (KSPOT)

Повна назва: Known Solution Peak of Tension Test.

Принцип: Фахівець ЗНАЄ правильну відповідь серед кількох альтернатив. Пік реакції повинен відповідати «правильній» альтернативі.

Умови:

  • Фахівець ПОВИНЕН бути впевнений у достовірності відомої відповіді.
  • Досліджуваний НЕ знає, яка альтернатива правильна.
  • Альтернативи рівно правдоподібні (5–7 варіантів).

Механізм:

  • Невинуватий: не знає правильного → реакції рівномірні.
  • Винуватий: знає правду → максимальна реакція саме на неї (орієнтовний рефлекс).

Приклад: Вкрадено 5000 грн. Альтернативи: 1000/2000/3000/5000/7000. Тільки злочинець дасть пік на 5000.

ℹ️ ПРИМІТКА: Порядок альтернатив варіюється між циклами для усунення позиційних ефектів.

K = Known (відомо). Фахівець знає правильну відповідь серед 5–7 альтернатив.

#22 Тест встановлення прихованої інформації (CIT)

  • Деталі події → категорії → 5–7 альтернатив (одна правдива, решта — відволікачі).

#8 7-позиційна федеральна система числового оцінювання

  • 0 = Нейтраль: реакції рівні або відсутні.
  • -1 = Можлива правда: невелика перевага К.
  • -2 = Правда: помітна перевага К.

#59 Процедура установки датчиків

Загальне правило: Неправильне розміщення = неякісний сигнал = ненадійні результати.

Послідовність:

  • Крок 3 — ЕДА: Два електроди на кінчики пальців (безіменний та середній). Без лосьйону.

#20 Професійна етика поліграфолога

Основні принципи:

  • Конфіденційність: результати не розкриваються без законних підстав.
  • Неупередженість: без особистого інтересу в результаті.
  • Добровільна участь: відмова від будь-якого примусу.
  • Точність: правдиве представлення можливостей та обмежень поліграфа.
  • Компетентність: тільки в межах кваліфікації.
  • Захист гідності: поважне ставлення незалежно від підозр і результатів.

Заборонені дії:

  • Маніпулювання для отримання «бажаного» результату.
  • Розкриття результатів без правових підстав.
  • Перебільшення точності поліграфа.

#69 Значення упередженості в поліграфологічних дослідженнях

Упередженість: Вплив попередніх переконань або очікувань фахівця на інтерпретацію результатів.

Джерела:

  • Гіпотеза попередньо відома («ми вважаємо, що це він»).
  • Репутація досліджуваного. Особиста симпатія/антипатія. Очікування замовника.

Методи мінімізації:

  • «Сліпе» оцінювання: фахівець не знає гіпотези замовника.
  • Комп'ютерна обробка (PolyScore, ASIT).
  • Незалежний повторний аналіз (peer review).
  • Числова система: алгоритмічний розрахунок знижує суб'єктивізм.
⚠️ УВАГА: «Підтверджувальне упередження» (confirmation bias) — найбільша загроза для об'єктивності поліграфолога.

#78 Оформлення та порядок використання отриманих результатів

  • Дата, час, місце. Дані фахівця (ПІБ, кваліфікація, ліцензія). Дані досліджуваного.
