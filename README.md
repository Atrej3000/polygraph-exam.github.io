## 🧠 Методологія

Посібник розроблений на основі доказових технік запам'ятовування ([Dunlosky et al., 2013](https://doi.org/10.1177/1529100612453266)):

| Техніка | Що робить | Де в посібнику |
|---|---|---|
| **Skeleton Map** | Створює когнітивну «файлову систему» для нового матеріалу | Навігаційна карта зверху |
| **Active Recall** | Тестування пам'яті замість перечитування (+50–80% retention) | 🧠 Блоки з прихованими відповідями |
| **Мнемоніки** | Компресія 20–30 фактів у 5–6 chunks | 🎯 Акроніми та асоціації |
| **Teach-back** | Вербалізація активує глибше кодування | ⏱️ «Поясни за 60 секунд» |
